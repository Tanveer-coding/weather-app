About This Weather App

Weather Dashboard

SkyCast is a simple weather dashboard built using HTML, CSS and JavaScript.

Live Weather API

The application uses the OpenWeatherMap API to fetch real-time weather data.

Dynamic Interface

Weather data updates dynamically using JavaScript DOM manipulation.

Features

  • Search weather of any city
  • Real-time temperature
  • Humidity information
  • Wind speed details
  • Sunrise & sunset time
  • Weather data of other cities
