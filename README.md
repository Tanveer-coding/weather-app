# 🌦️ Live Weather App

A responsive and interactive Live Weather Application built using **HTML, CSS, and JavaScript**. This project allows users to check real-time weather information for different cities with a clean and user-friendly interface.

## 🔗 Live Demo

[Click Here to Open the Weather App](https://weather-app-opal-rho-60.vercel.app/)

---

# 📌 Features

* 🌍 Search weather by city name
* 🌡️ Displays current temperature
* ☁️ Shows weather condition
* 💧 Humidity information
* 🌬️ Wind speed details
* 📱 Responsive design for mobile and desktop
* 🎨 Attractive UI with modern styling

---

# 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and responsive design
* **JavaScript** – Fetching and displaying weather data
* **Weather API** – Real-time weather information
* **Vercel** – Deployment platform
